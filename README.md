# 📚 Library Management System

A production-quality Full Stack Library Management System built with **ASP.NET Core Web API (.NET 10)** following **N-Layer Architecture**, modern software engineering principles, and enterprise-level best practices.

> This project was designed as both a university graduation project and a portfolio project to demonstrate backend architecture, clean code, security, and scalable application design.

---

# 🚀 Tech Stack

## Backend

- ASP.NET Core Web API (.NET 10)
- SQL Server
- Entity Framework Core (Code First)
- ASP.NET Identity
- JWT Authentication
- Refresh Tokens
- Google Authentication
- Email Verification
- OTP Authentication
- AutoMapper
- FluentValidation

# 🏗️ Architecture

This project follows **N-Layer Architecture**.

```
Client

↓

API Layer

↓

Application Layer

↓

Infrastructure Layer

↓

SQL Server
```

Projects

```
LibraryManagement.API

LibraryManagement.Application

LibraryManagement.Domain

LibraryManagement.Infrastructure

LibraryManagement.Shared
```

# ⚙️ Getting Started

## Clone

```bash
git clone https://github.com/YOUR_USERNAME/LibraryManagementSystem.git
```

---

## Restore Packages

```bash
dotnet restore
```

---

## Update Database

```bash
dotnet ef database update
```

---

## Run

```bash
dotnet run --project src/LibraryManagement.API
```

---

## Swagger

```
http://localhost:5132/swagger
```
